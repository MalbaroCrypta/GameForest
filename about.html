            FAQ
            Дані, легальність, оновлення.
            Як збираємо дані?
            Офіційні API + парсинг сторінок із кешем; демо працює статично.
            Це легально?
            Демо не продає товари. Для продакшену — тільки офіційні інтеграції та партнерські угоди.
            Точність 100%?
            Цілі — свіжі ціни та алерти. Дані кешуються, валідуються, є стор-відхилення.
            Як часто оновлюємо?
            У демо — статично. У продакшену — щогодини для цін, миттєво для алертів.
            Звідки ціни?
            Steam, PlayStation Store, Xbox, Switch, офіційні магазини ключів. Без «сірих» рітейлерів.
            Що з API?
            REST + вебхуки для алертів. Доступ до price history, best-deal, watchlists (B2B).
