Про GameForest
Що будуємо, для кого, монетизація та плани.

Що це?

Єдина панель для аналітики ігор, best-deal, compare та витягування метрик для бізнесу.

Для кого

Паблішери, студії, ком'юніті-менеджери, інвестори, B2B-партнери.

Цінність

Швидкий огляд, партнерські інструменти, API, топ-up канал з прозорою економікою.

Монетизація

  • B2B підписка, white-label.
  • API доступ до динаміки цін, скидок, compare.
  • Marketplace fee (комісія) після комплаєнсу.
  • Affiliate / партнерки.
